8. 编辑器和集成开发环境¶

有很多支持Python编程语言的集成开发环境。大多数编辑器和集成开发环境支持语法高亮，调试工具和 PEP 8 检查。

8.1. IDLE --- Python 编辑器和 shell¶

IDLE 是 Python 的集成开发和学习环境并通常会与 Python 安装版捆绑发布。 如果你在使用 Linux 并且尚未安装 IDLE 请参阅 在 Linux 上安装 IDLE。 更多信息请参阅 IDLE 文档。

8.2. 其他编辑器和 IDE¶

Python 的社区维基上有由社区成员提交的编辑器和 IDE 相关信息。 请访问 Python 编辑器 和 集成开发环境 以获取完整的列表。
